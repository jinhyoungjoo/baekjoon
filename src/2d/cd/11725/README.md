# 11725
## Find the parent of a tree.

- [Problem Link](https://www.acmicpc.net/problem/11725)
- Solutions: [C++](./11725.cpp)

### Problem Description
Given a tree, find the parent node of each node given the root.

### Algorithm
- Apply a graph traversal algorithm (e.g. BFS) to the given graph.
- In each step, when a new vertex is visited, record its parent as the original adjacent vertex.

### Keywords
- Graph
- Tree
- Graph Traversal
