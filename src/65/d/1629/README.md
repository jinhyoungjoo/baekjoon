## Multiplication

- [Problem Link](https://www.acmicpc.net/problem/1629)
- Solutions: [C++](./1629.cpp)

### Problem Description
Given integers A, B, and C, print out (A ^ B) % C.

### Algorithm
- Solve (A ^ B) % C by solving the subsolution (A ^ (B/2)) % C.
