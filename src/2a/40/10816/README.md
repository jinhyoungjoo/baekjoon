# 10816
## Number Card 2

- [Problem Link](https://www.acmicpc.net/problem/10816)
- Solutions: [C++](./10816.cpp)

### Problem Description
Given N cards and M integer queries, for each query print the number of cards where the number on the card is equal to the query.

### Algorithm
- Use an hash map to record the number of occurences for each integer.

### Keywords
- Sorting
- Binary Search
- Hash
